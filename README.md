# 💸 Discount Calculator (Python)

A simple Python program that calculates the final bill amount after applying discounts based on the total purchase value.

# 💡 Overview

This project was built as part of my Python learning journey.
It demonstrates how conditional statements can be used to apply different discount rates based on price ranges.

# ⚙️ Features
* 💰 Calculates discount based on amount
* 📊 Multiple discount ranges
* 🧾 Displays final payable amount
* ⚡ Simple and fast execution
# 🛠️ Technologies Used
* Python
# 📚 What I Learned
* Using if-elif-else conditions
* Implementing range-based logic
* Performing percentage calculations
* Building real-world use cases
# 🎯 Discount Rules
* ₹0 – ₹999 → No discount
* ₹1000 – ₹4999 → 10% discount
* ₹5000 – ₹9999 → 20% discount
* ₹10000 and above → 30% discount
# 🎯 Purpose
This project was created to practice conditional logic and simulate a real-world billing system.
